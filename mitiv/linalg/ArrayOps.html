Class ArrayOps

  • java.lang.Object
    • mitiv.linalg.ArrayOps

  • public class ArrayOps
extends java.lang.Object
    Implementation of optimized array operations. The main purpose of this class is to collect basic vectorized operations (static methods) which are useful elsewhere. TODO: The code for most static methods should be automatically written from template code (easier maintenance and less bugs).
