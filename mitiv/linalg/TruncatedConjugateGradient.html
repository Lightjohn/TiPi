      • solveQuadratic

        public static int solveQuadratic(double a,
                 double b,
                 double c,
                 double[] x)
        Compute the roots of a 2nd degree polynomial. Solve the quadratic equation: a*x^2 + b*x + c = 0 for real x. The number n of distinct real roots is returned and the roots if any are stored into x. If there is no roots, the contents of x is left unchanged; otherwise (n = 1 or 2) the two values of x are set with the roots in ascending order, i.e. such that x[0] <= x[1].
        Parameters:
        a - 2nd degree coefficient
        b - 1st degree coefficient
        c - 0th degree coefficient
        x - 2-element output array to store the result
        Returns:
        The number n of distinct real roots. If n >= 1, then x[0] <= x[1]; otherwise (n = 0) and x contents is left unchanged.
