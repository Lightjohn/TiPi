      • _apply

        protected void _apply(Vector vec,
          int job)
        Implement the operations performed by a linear operator in-place. This protected method is called by the `apply` method after checking of the arguments. Operation is done in-place which is only possible for endomorphisms. If this method is not overridden by the derived class, a default implementation which uses a temporary vector is provided.
        Parameters:
        vec - - The source and destination vector.
        job - - The type of operation to perform.
        Throws:
        NotImplementedException - Operation is not supported.
