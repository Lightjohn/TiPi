mitiv.linalg

Class Vector

  • java.lang.Object
    • mitiv.linalg.Vector
  • Direct Known Subclasses:
    ShapedVector

    public abstract class Vector
extends java.lang.Object
    A Vector is an element of a VectorSpace. An instance of this class is a collection of real values which can be addressed individually and for which it makes sense to apply the operations supported by the elements of a vector space (inner product, linear combination, etc.).

    At this level of abstraction nothing more has to be known. For instance, the restriction that indexed elements of a vector are reals is not an issue for complex valued vectors (a complex is just two reals).
