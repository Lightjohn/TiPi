Class InternalBuffer

  • java.lang.Object
    • mitiv.io.InternalBuffer

    public class InternalBuffer
extends java.lang.Object
    Class for data input/output buffer. An InternalBuffer is a small #ByteBuffer designed for efficient input/output of binary data.
    Author:
    Éric Thiébaut
