      • computeScalingFactors

        public static double[] computeScalingFactors(double dataMin,
                             double dataMax,
                             double fileMin,
                             double fileMax)
        Compute scaling factors SCALE and BIAS.

        Scaling factors SCALE and BIAS are used to convert data values into scaled values suitable to be stored in integers. To convert integer value fileValue into a data value dataValue, the formula is: 

             dataValue = SCALE*fileValue + BIAS
        the reciprocal formula is: 
             fileValue = round((dataValue - BIAS)/SCALE)
        Parameters:
        dataMin - - The minimum data value.
        dataMax - - The maximum data value.
        fileMin - - The minimum file value.
        fileMax - - The maximum file value (should be strictly greater than dataMin).
        result - - An array of 2 doubles to store SCALE and BIAS (in that order).
