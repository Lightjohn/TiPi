        public double[] flatten(boolean forceCopy)
        Description copied from interface: DoubleArray
        Flatten the array of double's in a simple array.

        The contents of a (multi-dimensional) DoubleArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the DoubleArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the DoubleArray object is multi-dimensional, the storage of the returned result is column-major order.

        Specified by:
        flatten in interface DoubleArray
        Parameters:
        forceCopy - - Set true to force a copy of the internal data even though it can already be in a flat form.
        Returns:
        A simple array of double's with the contents of the DoubleArray.

      • flatten

        public double[] flatten()
        Description copied from interface: DoubleArray
        Flatten the contents of double's in a simple array.

        The contents of a (multi-dimensional) DoubleArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the DoubleArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the DoubleArray object is multi-dimensional, the storage of the returned result is column-major order.

        Depending on the storage layout, the returned array may or may not share the same storage as the DoubleArray array. Call flatten(true) to make sure that the two storage areas are independent.

        Specified by:
        flatten in interface DoubleArray
        Returns:
        A simple array of doubles with the contents of the DoubleArray array.
        See Also:
        #flatten(boolean)}, {@link Shaped#COLUMN_MAJOR}.
