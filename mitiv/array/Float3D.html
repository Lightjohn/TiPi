      • flatten

        public float[] flatten(boolean forceCopy)
        Description copied from interface: FloatArray
        Flatten the array of float's in a simple array.

        The contents of a (multi-dimensional) FloatArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the FloatArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the FloatArray object is multi-dimensional, the storage of the returned result is column-major order.

        Specified by:
        flatten in interface FloatArray
        Parameters:
        forceCopy - - Set true to force a copy of the internal data even though it can already be in a flat form.
        Returns:
        A simple array of float's with the contents of the FloatArray.

      • flatten

        public float[] flatten()
        Description copied from interface: FloatArray
        Flatten the contents of float's in a simple array.

        The contents of a (multi-dimensional) FloatArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the FloatArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the FloatArray object is multi-dimensional, the storage of the returned result is column-major order.

        Depending on the storage layout, the returned array may or may not share the same storage as the FloatArray array. Call flatten(true) to make sure that the two storage areas are independent.

        Specified by:
        flatten in interface FloatArray
        Returns:
        A simple array of floats with the contents of the FloatArray array.
        See Also:
        #flatten(boolean)}, {@link Shaped#COLUMN_MAJOR}.

      • wrap

        public static Float3D wrap(float[] data,
           int offset,
           int stride1,
           int stride2,
           int stride3,
           int dim1,
           int dim2,
           int dim3)
        Wrap an existing array in a 3D array of float's with given dimensions, strides and offset.

        This creates a 3D array of dimensions {dim1,dim2,dim3} sharing (part of) the contents of data in arbitrary storage order. More specifically: 

        arr.get(i1,i2,i3) = data[offset + stride1*i1 + stride2*i2 + stride3*i3]
        with arr the returned 3D array.
        Parameters:
        data - - The array to wrap in the 3D array.
        offset - - The offset in data of element (0,0,0) of the 3D array.
        stride1 - - The stride along the 1st dimension.
        stride2 - - The stride along the 2nd dimension.
        stride3 - - The stride along the 3rd dimension.
        dim1 - - The 1st dimension of the 3D array.
        dim2 - - The 2nd dimension of the 3D array.
        dim3 - - The 3rd dimension of the 3D array.
        Returns:
        A 3D array sharing the elements of data.
