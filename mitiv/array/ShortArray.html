      • flatten

        short[] flatten(boolean forceCopy)
        Flatten the array of short's in a simple array.

        The contents of a (multi-dimensional) ShortArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the ShortArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the ShortArray object is multi-dimensional, the storage of the returned result is column-major order.

        Parameters:
        forceCopy - - Set true to force a copy of the internal data even though it can already be in a flat form.
        Returns:
        A simple array of short's with the contents of the ShortArray.

      • flatten

        short[] flatten()
        Flatten the contents of short's in a simple array.

        The contents of a (multi-dimensional) ShortArray can be stored in many different forms. This storage details are hidden to the end-user in favor of a unified and comprehensive interface. This method returns the contents of the ShortArray object as a simple flat array, i.e. successive elements are contiguous and the first element has 0-offset. If the ShortArray object is multi-dimensional, the storage of the returned result is column-major order.

        Depending on the storage layout, the returned array may or may not share the same storage as the ShortArray array. Call flatten(true) to make sure that the two storage areas are independent.

        Returns:
        A simple array of shorts with the contents of the ShortArray array.
        See Also:
        #flatten(boolean)}, {@link Shaped#COLUMN_MAJOR}.
