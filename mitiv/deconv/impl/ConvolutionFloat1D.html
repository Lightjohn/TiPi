    public class ConvolutionFloat1D
extends WeightedConvolutionOperator
    Implements a FFT-based weighted convolution for 1D arrays of float's.

    It is recommended not to directly instantiate this class but rather use one of the factory methods of the parent class WeightedConvolutionOperator.build(). Have a look at the documentation of WeightedConvolutionOperator for a description of what exaclty does this kind of operator.
