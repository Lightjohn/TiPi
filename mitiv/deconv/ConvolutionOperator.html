    public class ConvolutionOperator
extends ShapedLinearOperator
    Implements a FFT-based convolution.

    The convolution operator H writes: 

       H = F'.diag(F.h).F
    with F the FFT (Fast Fourier Transform) operator and h the PSF (Point Spread Function).

      • ConvolutionOperator

        public ConvolutionOperator(RealComplexFFT FFT,
                   Vector psf,
                   Vector mtf)
        Create a new FFT-based convolution operator given the PSF or the MTF.

        At least one of the point spread function (PSF) or the modulation transfer function (MTF) must be given (that is non-null). If mtf is not null it is used as the MTF; otherwise the FFT operator is applied to psf to compute the MTF. Note that if the MTF is directly provided, a simple reference to mtf is kept by the operator. Thus, you may have to clone mtf if you intend to modify its contents while using the operator.

        Parameters:
        FFT - - The Fast Fourier Transform operator.
        psf - - The point spread function (PSF) or null.
        mtf - - The modulation transfer function (MTF) or null. - If true, h is the MTF; otherwise, h is the PSF. Note that if h is the MTF, a simple reference to it is kept by the operator. Thus, you should clone h if you intend to modify its contents while using the operator.
