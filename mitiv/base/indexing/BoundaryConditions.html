      • buildIndex

        public static final void buildIndex(int[] index,
              int offset,
              BoundaryConditions condition)
        Store index values with given offset and boundary conditions.

        This static method stores the shifted index: 

             index[j] = f(j + offset);
        where f(k) depends on the boundary condition and maps index k in {0, 1, ..., N-1} with N = index.length is the length of the considered dimension.

        See #buildIndex(int, int, int) for examples.

        Parameters:
        index - - An array to store the result.
        offset - - The offset of positions.
        condition - - The boundary conditions (PERIODIC, or MIRROR, otherwise NORMAL).

      • buildIndex

        public static final int[] buildIndex(int length,
               int offset,
               BoundaryConditions condition)
        Create an array of index values with given offset and boundary conditions.

        For instance: 

             buildIndex(7, -1, NORMAL) -----> {0, 0, 1, 2, 3, 4, 5}
     buildIndex(7, -2, PERIODIC) ---> {5, 6, 0, 1, 2, 3, 4}
     buildIndex(7, -2, MIRROR) -----> {2, 1, 0, 1, 2, 3, 4}
        Parameters:
        length - - The length of the dimension.
        offset - - The offset along the dimension.
        condition - - The boundary conditions (PERIODIC, or MIRROR, otherwise NORMAL).
        Returns:
        An array of indexes in the set {0, 1, ..., length-1}.
