      • getElapsedTime

        public double getElapsedTime()
        Get elapsed time in seconds.

        This method yields the total elapsed time integrated by the timer. If the timer has been stopped the elapsed time will not change and is simply given by the sum of the elapsed times between the last call to start() (denoted as start below) and the next call to stop() (denoted as stop1 below) and between the successive calls to resume() and stop() since then (denoted as resumek and stopk respectively for the k-th call). The formula writes: 

           (stop1 - start) + (resume1 - stop2) + ... + (resumen-1 - stopn)
        If the timer is running the elapsed time between the current time (denoted as current below) and the last call to resume() (denoted as resumen below) is added, the formula becomes: 
           (stop1 - start) + (resume1 - stop2) + ... + (resumen - current)
        Thus the same timer can be used to integrate execution time.

        Returns:
        The elapsed time in seconds.
