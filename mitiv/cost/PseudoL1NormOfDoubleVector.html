      • applyProx

        public void applyProx(Vector out,
             double alpha,
             Vector inp,
             double tol)
        Description copied from interface: ProximalOperator
        Apply the proximal operator of a cost function. The purpose of this method is to find the minimum of: 
         alpha*f(out) + (1/2)*norm2(out - inp)
        with respect to vector out and given weight alpha and the input vector out. The argument tol is a relative tolerance parameter to find an approximate solution to the problem.
        Parameters:
        out - - The vector of output variables.
        alpha - - A non-negative weight for the cost.
        inp - - The vector of input variables.
        tol - - A non-negative tolerance parameter.
