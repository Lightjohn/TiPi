        public void setRho(double[] beta)
        Compute the modulus ρ on a Zernike polynomial basis

        The coefficients β are normalized and the modulus is ρ = Σ_n β_n Z_n

        Parameters:
        beta - Zernike coefficients

      • setPhi

        public void setPhi(double[] alpha)
        Compute φ the part of the phase of the pupil function on a Zernike polynomial basis

        φ = Σ_n α_n Z_{n+3}

        Parameters:
        alpha - Zernike coefficients

      • setDefocus

        public void setDefocus(double[] defocus)
        Parameters:
        defocus - Update the defocus and the depth functions according the parameters defocus. Depending on the number of elements of defocus: 4 : defocus = {nu_s, c_x, c_y, \nu_i} 3 : defocus = {c_x, c_y, \nu_i} 2 : defocus = {\nu_s, \nu_i} 1 : defocus = {\nu_i} where \NU_S : wavenumber in specimen medium (n_s/lambda) \NU_I : wavenumber in specimen medium (n_i/lambda) [c_y, c_x] : center of both defocus and depth function
