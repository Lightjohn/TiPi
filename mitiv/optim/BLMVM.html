mitiv.optim

Class BLMVM

  • java.lang.Object
    • mitiv.optim.ReverseCommunicationOptimizer
      • mitiv.optim.BLMVM

  • public class BLMVM
extends ReverseCommunicationOptimizer
    Multivariate non-linear optimization with simple bound constraints by BLMVM method.

    There are some differences compared to LBFGS, the unconstrained version of the algorithm:

    1. The initial variables must be feasible. This is easily achieved by applying the projector on the initial variables.
    2. The gradients computed by the caller are projected. This means that they are not left unchanged.
    3. The line search procedure should only implement a sufficient decrease test (e.g. first Wolfe condition).

    Author:
    Éric Thiébaut.
