Class BoundProjector

    public abstract class BoundProjector
extends ConvexSetProjector
    Abstract class for bound projection.

    A BoundProjector can be used to implement bound constraints. Compared to a ConvexSetProjector, it has also the ability to project the gradient, that is to generate (the opposite) of a feasible steepest descent direction.

        public void projectDirection(Vector x,
                    Vector d,
                    boolean ascent,
                    Vector dp,
                    double[] bnd)
        Project a direction to the null-space of the linearized constraints.
        Parameters:
        x - - The variables (in principle they should be feasible).
        d - - The direction to be projected.
        ascent - - If true d is an ascent direction, otherwise d is a descent direction. Typically, one set ascent true if d is the gradient.
        dp - - The projected direction (can be the same as d).
        bnd - - An array of two values used to store step bounds, the first value is the ``free path length'' (the largest step length which can be taken along the projected direction without crossing any bounds), the second value is the smallest step length above which the variables will remain unchanged because of bounds. If ascent is false, then the line search seeks for improved variables as: x + alpha*d; otherwise, the line search seeks for improved variables as: x - alpha*d. In both cases alpha is the nonnegative step length to which the computed bounds apply.

        Protected method to project a direction to the null-space of the linearized constraints.

        This abstract method must be overridden by instantiable sub-classes, it is guaranteed to be called with checked arguments. The input and output vectors can be the same.

        Parameters:
        x - - The variables (they belongs to the feasible space).
        d - - The input direction to be projected.
        ascent - - If true `d` is an ascent direction, otherwise `d` is a descent direction.
        dp - - The projected direction.
        bnd - - Bounds for the step length.
