      • findStepLimits

        public double[] findStepLimits(Vector vec,
                      Vector dir,
                      int orient)
        Description copied from class: BoxedSet
        Find the limits of the step size.

        Along the search direction the new variables are computed as: 

         proj(x +/- alpha*d)
        where proj is the projection onto the feasible set, alpha is the step length and +/- is a plus for a descent direction and a minus otherwise. This method computes 3 specific step lengths: smin1 which is the step length for the first bound reached by the free variables along the search direction; smin2 which is the step length for the first bound reached by the free variables along the search direction and with a non-zero step length; smax which is the step length for the last bound reached by the free variables along the search direction.

        In other words, for any 0 <= alpha <= smin1, no free variables may exceed a bound, while, for any alpha >= smax, the projected variables are the same as those obtained with alpha = smax. If d has been properly projected (e.g. by BoxedSet.projectDirection(mitiv.linalg.Vector, mitiv.linalg.Vector, mitiv.linalg.Vector, int)), then smin1 = smin2 otherwise 0 <= smin1 <= smin2 and smin2 > 0.

        Specified by:
        findStepLimits in class BoxedSet
        Parameters:
        vec - - The current variables.
        dir - - The search direction.
        orient - - The orientation of the search direction (see BoxedSet.projectDirection(mitiv.linalg.Vector, mitiv.linalg.Vector, mitiv.linalg.Vector, int)).
        Returns:
        A 3-element array of values: {smin1,smin2,smax}.
