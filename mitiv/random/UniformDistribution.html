      • setState

        public void setState(double a,
            double b)
        Set the bounds of the uniform distribution.

        The arguments are the bounds of the uniform distribution. The inferior bound will be set to the least of a and b, while the superior bound will be set to the greatest of a and b. Note that, unless a = b, these bounds are never reached by the pseudo-random values generated by nextFloat() nor nextDouble().

        Parameters:
        a - - One bound of the uniform distribution.
        b - - Other bound of the uniform distribution.
